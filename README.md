# Welcome Card Demo

A simple, interactive React demo showcasing a centered welcome card with:

- Dynamic title update via text input
- Background color switch (Red / Green)
- Responsive design with mobile-friendly spacing
- Modern card elevation with custom box-shadow
- Smooth transitions and hover effects

Built with **React + Vite + TypeScript** and plain CSS (no external UI libraries).

## Live Demo

https://welcome-card-demo.vercel.app/

## Features

- Real-time title update in the header
- Click buttons to change the entire page background
- Carefully crafted box-shadow for realistic card elevation (strong bottom-right + subtle ambient)
- Extra margin on mobile screens for better usability
- Clean, maintainable code with clear comments

## Tech Stack
